# Linux Commands Project

This repository contains commonly used Linux commands that are essential for Linux Administrators and beginners.

## Topics Covered

### File Management Commands
- ls – list files and directories
- cd – change directory
- pwd – print working directory
- mkdir – create directory
- rm – remove files or directories
- cp – copy files
- mv – move or rename files

### User Management Commands
- useradd – create new user
- passwd – set password for user
- userdel – delete user
- groupadd – create group
- groupdel – delete group

### Permission Commands
- chmod – change file permissions
- chown – change file ownership

### Process Management
- ps – view running processes
- top – monitor system processes
- kill – terminate process

### Networking Commands
- ping – check network connectivity
- ifconfig / ip addr – check IP address
- netstat – view network connections

## Project Purpose

The purpose of this project is to practice and document important Linux commands used in real-world system administration.
